## Trabalho da disciplina de LP5

 Este repositório consiste na aplicação dos seguintes design patterns em um mini sistema escrito em Java: Singleton, Factory Method, Abstract Factory, Bridge, Decorator e State. Será desenvolvido um mini sistema de hamburgueria paar registrar pedidos de clientes.Na etapa de planejamento do prjeto também será escrito um diagrama UML de classes e um de estado
 
 Stack do projeto:

- Spring boot 4.0.5
- Java 25
- Maven
